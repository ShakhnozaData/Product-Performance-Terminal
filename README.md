# Product Performance Terminal (Power BI)

Interactive dashboard designed to track $25.12M in global sales and optimize product markups.

### 📊 [View Live Interactive Dashboard on novyPro](https://www.novypro.com/create_project/executive-sales--product-performance-terminal)

### Key Insights
- **Profitability:** Clothing & Accessories drive 40% of total profit.
- **Optimization:** Identified high-volume/low-markup units for price adjustments.
- **Growth:** Sustained 16.79% YoY growth with major peaks in May and December.

### Tools Used
- **Power BI:** Star schema modeling & DAX.
- **UI Design:** Custom Dark-Mode Terminal interface.
